Mobile-Regeln die wir einhalten

Layout
✓ Alle Grids → 1 Spalte unter 768px
✓ Schrift min. 16px (kein Zoom nötig)
✓ Touch-Targets min. 44×44px
✓ Kein horizontales Scrollen
Performance
✓ Bilder mit loading="lazy"
✓ Slider-Bilder: max 800px Breite
✓ Kein externe JS-Libraries
✓ CSS-first, JS nur wo nötig
Navigation
✓ Hamburger ab 768px
✓ Dropdown → vertikale Liste
✓ Telefon-Link immer klickbar
✓ CTA sticky am unteren Rand
Slider Mobile
✓ Touch-Events (touchstart/move)
✓ Pointer Events API (unified)
✓ Handle vergrößert (44px)
✓ Passiver Event-Listener (kein Scroll-Block)
